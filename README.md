# CarCatalog
Race car catalog developed with NodeJS and AngularJS

##Prerequisites
To Install and run the project you need to have installed globally Bower and NodeJS.
``` 
https://bower.io
https://nodejs.org/en
```

##Installation

```
[sudo] npm install  
bower install
```

##Usage
Run the server  
```
node server.js
```

##View
The Server is running on http://localhost:3000
